# **App Name**: Micheal Joseph Portfolio

## Core Features:

- Responsive Header: Sticky header with responsive navigation for About, Case Studies, Expertise, and Contact.
- Hero Section: Hero section featuring a headline, UVP, and CTA button linking to case studies.
- Analyst-Marketer Narrative: Concise 'About Me' narrative emphasizing a data and ROI focus.
- Interactive Case Studies: Interactive case study hub that uses chart.js
- Expertise Grid: A clean, four-column grid categorizing skills to demonstrate structured thinking.
- Dynamic Data Visualization: Display data visualizations from the case studies, dynamically chosen using a button click, displayed using Chart.js.
- Final CTA: A concise call to action with contact info and a LinkedIn social link.

## Style Guidelines:

- Primary background: Vibrant brown (#A67B5B) for a warm and inviting feel.
- Text color: Deep gold (#D4A27A) for elegance and sophistication.
- Accent color: Light gold (#E5CAA6) for CTAs, links, and chart highlights.
- Headings font: 'Lora' serif for authority.
- Body text font: 'Inter' sans-serif for readability.
- Minimal use of icons to highlight skills in the expertise section.
- Bold Minimalism: Clean, spacious layout maximizing data impact.
